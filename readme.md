# Инструкция по запуску
* Скачать и установить интерпретатор PHP 
* Скачать [OpenServer](https://ospanel.io/download/) или другую подобную программу
* Запустить OpenServer и в папку домена залить файл index.php 
* Открыть браузер и в поисковой строку написать "Localhost"
